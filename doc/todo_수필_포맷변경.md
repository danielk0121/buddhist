# 수필 포맷 변경 작업

수필 45종의 `## 이야기` 섹션 앞 원문 구절 부분을 새 포맷으로 교체한다.

---

## 변경 내용

### 변경 전

```
## 이야기

관자재보살 행심반야바라밀다시 조견오온개공 도일체고액 (觀自在菩薩 行深般若波羅蜜多時 照見五蘊皆空 度一切苦厄)

어느 날, 관자재보살이 깊은 명상에 잠겼습니다.
```

### 변경 후

````
## 한자 풀이

```

## 한자 구절 한국어 풀이

관자재보살이 깊은 반야바라밀다를 행할 때, 오온이 모두 공함을 환히 비추어 보고, 온갖 고통과 재앙을 건넜다.

## 이야기

어느 날, 관자재보살이 깊은 명상에 잠겼습니다.
````

---

## 작업 규칙

- `## 단어 사전` 섹션은 그대로 유지
- 기존 `## 이야기` 섹션 첫 줄의 원문 구절(한글음독 + 한자)을 삭제
- 그 자리에 `## 한자 풀이` 섹션 삽입 (전체를 마크다운 코드 블록으로 감쌈):
  - 한문 원문 (구절 단위 줄바꿈)
  - 한글 음독 (한문과 같은 줄바꿈 대응)
  - 빈 줄
  - 각 한자를 `한자 음 (뜻)` 형식으로 나열, 어절 단위로 빈 줄로 묶음
- `## 한자 구절 한국어 풀이` 섹션 삽입:
  - 해당 구절의 자연스러운 한국어 해설 (1~3문장)
- `## 이야기` 섹션은 본문 이야기만 남김
