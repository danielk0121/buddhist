# 수필 문서 원문 전체 보기 섹션 추가

## 개요

각 수필 문서 최하단 `## 읽고 나서` 섹션 아래에 `## 원문 전체 보기` 섹션을 추가한다.

---

## 추가할 섹션 형식

````markdown
## 원문 전체 보기

```
(한자 원문 전체)
(한글 음독 전체)

(각 한자 음 뜻 풀이 전체 — 어절 단위로 빈 줄 구분)
```
````

- 한자 원문과 음독은 수트라 문서(`src/sutra/`)의 `## 원문 (한문)`, `## 원문 한글 음독`에서 가져옴
- 각 한자 풀이는 `한자 음 (뜻)` 형식으로 어절 단위 빈 줄 구분
- 수트라 문서에 한자 풀이가 없는 경우 새로 작성
