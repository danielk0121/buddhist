# 저작권 분석

이 프로젝트에서 경전 문서를 작성할 때 고려해야 할 저작권 사항을 정리합니다.

---

## 원전(原典)의 저작권

### 한문 원전

- 한문 원전(현장 역, 구마라집 역 등)은 모두 수백 년 전 작성된 공유 저작물(퍼블릭 도메인)이다.
- 저작권 보호 기간(사후 70년)이 만료되었으므로 자유롭게 인용·수록이 가능하다.

### 팔리어 원전

- 팔리어 정전(Pāli Canon)은 기원전~수백 년 전 성립된 공유 저작물이다.
- 팔리 성전 협회(PTS, Pali Text Society)의 편집본은 편집 저작권이 있을 수 있으나, 원문 자체는 공유 저작물이다.

---

## 번역문의 저작권

### 문제가 되는 경우

현대 번역문은 번역자의 창작물로서 저작권이 발생한다.

- 대한불교 조계종 표준 한글 번역본 — 조계종 저작권 보유. 무단 전재 불가.
- 불교 출판사(동국역경원 등)의 현대 번역본 — 출판사·번역자 저작권 보유.
- 개인 번역가의 번역본 — 번역자 저작권 보유.

### 이 프로젝트의 방침

- 기존 번역을 그대로 복사하지 않는다.
- 한문 원전을 직접 풀어쓰는 방식으로 독자적인 우리말 번역을 작성한다.
- 기존 번역을 참고하더라도 표현을 독립적으로 재구성하여 작성한다.
- 인용이 필요한 경우 출처를 명시하고, 인용 분량을 최소화한다.

---

## 다라니·진언의 저작권

- 다라니(陀羅尼)·진언(眞言)의 산스크리트어 원문은 공유 저작물이다.
- 한국식 음사(音寫 — 소리 음·베낄 사) 표기(예: 신묘장구대다라니)는 수백 년 전부터 전해진 것으로, 특정 현대 편집본의 표기를 그대로 사용할 경우 편집 저작권 주의가 필요하다.
- 일반적으로 통용되는 독송 표기는 공유 저작물로 간주되나, 특정 출판물의 표기를 그대로 사용하는 것은 피한다.

## 경전별 주의 사항

### 반야심경

- 현장(玄奘) 한역 원문 — 공유 저작물, 자유 인용 가능
- 조계종 표준 한글 반야심경(2011년 공포) — 조계종 저작권. 이 프로젝트에서는 독자 번역을 사용한다.

### 금강경

- 구마라집(鳩摩羅什) 한역 원문 — 공유 저작물, 자유 인용 가능
- 현대 번역본(동국역경원 등) — 저작권 있음. 독자 번역을 사용한다.

### 천수경

- 한문 원전 — 공유 저작물
- 신묘장구대다라니 통용 음사 — 일반적으로 통용되는 공유 표기 사용
- 조계종 표준 우리말 천수경 — 조계종 저작권. 독자 번역을 사용한다.

### 법화경·화엄경 등 대승경전

- 한문 원전 — 공유 저작물
- 동국역경원 번역본 — 저작권 있음. 독자 번역 또는 공유 번역본을 사용한다.

### 팔리어 경전

- 팔리어 원문 — 공유 저작물
- 각묵 스님, 전재성 번역본 등 — 번역자 저작권 있음. 독자 번역을 사용한다.
- 스리랑카 불교출판협회(BPS) 영역본 — 일부 공유 라이선스 적용본 존재, 확인 필요.

## 수필 재구성 방식(`_easy.md`)의 저작권 검토

이 프로젝트는 `01_반야심경_easy.md`를 시작으로, 경전을 수필처럼 이야기 형식으로 재구성하는 작업을 진행하고 있다. 이 방식은 기존 번역문과는 성격이 다르므로 별도로 검토한다.

### 저작권 위험도: 낮음

수필 재구성 방식은 저작권 관점에서 기존 번역 방식보다 오히려 안전하다. 이유는 다음과 같다.

- **원전 저본(底本)**: 한문 원전(공유 저작물)의 내용을 바탕으로 작성. 특정 현대 번역문을 저본으로 삼지 않는다.
- **창작적 변형**: 원문의 구성·순서·표현을 그대로 옮기지 않고, 서술자의 시점·비유·해설을 더해 완전히 새로운 글로 재창작한다. 이는 저작권법상 2차적 저작물(또는 독립 저작물)에 해당한다.
- **표현의 독자성**: "꽃 한 송이" 비유, 이야기 구조, 단어 사전 구성 등은 기존 어떤 번역본에도 없는 독자적 표현이다.

### 주의해야 할 지점

수필 재구성 방식에서도 주의할 부분이 있다.

**1. 핵심 구절의 처리**

경전의 핵심 구절("색즉시공 공즉시색", "아제 아제 바라아제..." 등)은 수필 안에서도 인용된다. 이 구절 자체는 공유 저작물(한문 원전)이므로 문제없다. 단, 구절의 우리말 풀이를 특정 현대 번역본의 표현과 동일하게 쓰는 것은 피한다.

- 현재 `01_반야심경_easy.md`의 주문 풀이 — "가세, 가세, 저 언덕으로 가세..." — 는 독자적으로 작성된 표현으로, 조계종 표준본의 표현과 다르다. 문제없음.

**2. 사실 정보와 교리 해설**

불교 교리(오온, 육근, 육경, 사성제 등)의 정의와 설명은 저작권 보호 대상이 아니다. 사실(fact) 자체에는 저작권이 없으며, 표현 방식에만 저작권이 생긴다. 단어 사전 형식의 용어 해설은 이 프로젝트의 독자적 표현이므로 문제없다.

**3. 구조적 유사성**

수필이 특정 기존 책(예: 법정 스님, 성철 스님의 경전 해설서 등)의 구성·서술 구조를 그대로 따라가면 저작권 문제가 생길 수 있다. 현재 작업은 이야기 전개와 비유를 독자적으로 구성하고 있으므로 문제없다.

## 결론 및 권고

### 기존 번역 방식(`src/XX_경전명.md`)

- 한문·팔리어 원전 인용은 자유롭게 가능하다.
- 현대 번역문은 반드시 독자적으로 재작성한다.
- 조계종·동국역경원 등 특정 기관의 번역을 그대로 수록하지 않는다.
- 참고한 번역이 있을 경우, 문서 하단에 참고 출처를 표기한다.
- 이 프로젝트의 번역문은 한문 원전을 저본(底本 — 밑 저·근본 본)으로 삼아 작성한 독립 저작물임을 명시한다.

### 수필 재구성 방식(`src/XX_경전명_easy.md`)

- 현재의 방식(한문 원전 저본, 독자적 이야기 구성, 비유와 해설 추가)은 저작권 침해 위험이 낮다.
- 핵심 구절의 우리말 풀이를 특정 현대 번역본과 동일하게 쓰지 않는다.
- 기존 경전 해설서(스님들의 법문집 등)의 구성·표현을 직접 차용하지 않는다.
- 각 `_easy.md` 파일 상단에 "한문 원전을 저본으로 한 독자 재구성"임을 명시하는 것이 좋다.

### GitHub 공개 레포 게시 판단

- 기존 번역 방식: 침해 가능성 낮음 (기존 판단 유지)
- 수필 재구성 방식: 침해 가능성 매우 낮음. 독자적 창작물로서 오히려 저작권 보호를 받는 방향.
- 권고: README에 "비영리·학습 목적, 한문 원전 저본의 독립 번역 및 재구성"임을 명시한다.
- 천수경 개경게 표현 주의 사항(기존 지적)은 여전히 유효하다. 독자적 표현으로 재작성 권고.
