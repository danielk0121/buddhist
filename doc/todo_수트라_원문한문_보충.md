# 수트라 원문 한문 보충 작업

`src/sutra/` 경전 문서 중 `## 원문 (한문)` 섹션에 짧은 인용 구절만 있는 파일에
인터넷 검색을 통해 실제 경전 한문 원문을 보충합니다.

## 기준

- **보충 대상**: 코드 블록에 1~3줄 짧은 구절만 있고 괄호 주석이 붙은 경우
- **목표**: 해당 경전의 대표적인 한문 원문 단락(10줄 이상)을 추가
- **출처**: 한문 대장경 원문 기준 (T대정신수대장경)
