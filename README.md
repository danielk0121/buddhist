# Buddhist

불경을 읽기 쉬운 현대 한국어로 정리하는 프로젝트입니다.

## 목적

한문·고어로 된 불경을 누구나 편하게 읽을 수 있도록 현대 우리말로 풀어 정리합니다.  
원문의 뜻을 최대한 살리면서도 일상 언어로 이해할 수 있게 작성하는 것을 목표로 합니다.

## 구조

```
buddhist/
├── src/        # 불경 텍스트 (마크다운)
├── doc/        # 용어 해설, 배경 설명 등 보조 문서
├── ref/        # 참고 자료
└── to_pdf/     # PDF 변환 관련 스크립트·설정
```

## 수록 예정 경전

- 반야심경 (般若心經)
- 금강경 (金剛經)
- 천수경 (千手經)
- 법화경 (法華經) — 일부 발췌
- 화엄경 (華嚴經) — 일부 발췌
