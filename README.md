# Buddhist

불경을 읽기 쉬운 현대 한국어로 정리하는 프로젝트입니다.

> **저작권 고지**
> 모든 문서는 한문·팔리어 원전(공유 저작물)을 저본으로 독립 작성한 번역·재구성물입니다.
> 조계종·동국역경원 등 현대 번역본을 그대로 수록하지 않습니다.
> 상세 내용: [doc/저작권_분석.md](doc/저작권_분석.md)

## 목적

한문·고어로 된 불경을 누구나 편하게 읽을 수 있도록 현대 우리말로 풀어 정리합니다.
원문의 뜻을 최대한 살리면서도 일상 언어로 이해할 수 있게 작성하는 것을 목표로 합니다.

## 구조

- `src/` — 불경 원본 문서를 마크다운(`.md`) 파일로 작성
- `doc/` — 용어 해설, 배경 설명 등 보조 문서
- `search/` — 참고 자료
- `to_pdf/` — `src/`의 내용을 PDF 파일로 변환하여 저장

## 디렉터리 규칙

- `src/` — 원본 문서 내용을 마크다운(`.md`) 파일로 작성한다.
- `to_pdf/` — `src/`의 내용을 PDF 파일로 변환하여 저장한다.

## 문서 작성 규칙

- 모든 문서는 한국어로 작성한다. 최대한 한국어를 사용한다.
- 한자가 언급되는 경우, 모든 한자에 대한 한국어 설명(훈음)을 추가한다.
- 표(table) 형태 대신 목록(list) 형태를 사용한다.

## 프로젝트 목표

부처님 직설 경전 전체를 현대 한국어로 정리하는 것을 목표로 합니다.
대승경전·아함경전·팔리어 경전 등 45종 경전 목록은 [doc/todo_경전.md](doc/todo_경전.md)에서 확인할 수 있습니다.
각 경전 문서에는 경전 소개·주요 구성·핵심 사상·핵심 개념 해설·경전의 의의 섹션이 포함되어 있습니다.
