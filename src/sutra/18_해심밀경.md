# 해심밀경 (解深密經)

해(풀 해)·심(깊을 심)·밀(빽빽할 밀)·경(경전 경)

> 해심밀경(解深密經)
> 당(唐) 현장(玄奘 — 검을 현·감출 장) 한역 (647년) | 현대 우리말 풀어쓰기

---

## 경전 소개

- **원어:** 산스크리트어 원전 『산디니르모차나 수트라(Saṃdhinirmocana Sūtra)』를 한문으로 번역
- **구성:** 5권 8품(品 — 품 품)
- **핵심 사상:** 유식(唯識 — 오직 유·알 식) 사상 — 모든 현상은 오직 마음의 식(識)이 만들어낸 것. 삼성(三性 — 셋 삼·성품 성)과 삼무자성(三無自性 — 셋 삼·없을 무·스스로 자·성품 성)
- **특징:** 유식학파(唯識學派)의 근본 경전. 아뢰야식(阿賴耶識 — 저 아·의지할 뢰·집 야·알 식)과 삼전법륜(三轉法輪 — 셋 삼·구를 전·법 법·바퀴 륜)을 설명

---

## 원문 (한문)

```
廣慧，若諸有情自無始世，種種界中，更互為緣，生起集聚，有情之類，
於此識中，有種種界差別流注。廣慧，阿陀那識甚深細，一切種子如瀑流，
我於凡愚不開演，恐彼分別執為我。
```

## 핵심 사상

### 삼성 (三性)

삼(셋 삼)·성(성품 성)

- **변계소집성(遍計所執性):** 변(두루 변)·계(셀 계)·소(바 소)·집(잡을 집)·성. 두루 헤아려 집착하는 성질 — 착각과 분별로 만들어진 허망한 모습
- **의타기성(依他起性):** 의(의지할 의)·타(다를 타)·기(일어날 기)·성. 다른 것에 의지해 일어나는 성질 — 인연(因緣)에 의해 생겨나는 현상
- **원성실성(圓成實性):** 원(둥글 원)·성(이룰 성)·실(참 실)·성. 원만하게 이루어진 참된 성질 — 있는 그대로의 진실

### 아뢰야식 (阿賴耶識)

아(저 아)·뢰(의지할 뢰)·야(집 야)·식(알 식)

- 모든 경험과 업(業 — 업 업)의 씨앗을 저장하는 근본 식(識)
- 장식(藏識 — 감출 장·알 식)이라고도 함 — 모든 것을 담아두는 창고
- 현행(現行 — 나타날 현·행할 행)과 종자(種子 — 씨 종·아들 자)가 서로 훈습(熏習 — 향기 훈·익힐 습)하며 순환

### 삼전법륜 (三轉法輪)

삼(셋 삼)·전(구를 전)·법(법 법)·륜(바퀴 륜)

- **초전법륜(初轉法輪):** 초(처음 초). 아함(阿含) — 사성제(四聖諦)와 무아(無我)를 설함
- **이전법륜(二轉法輪):** 이(두 이). 반야(般若) — 모든 법이 공(空)함을 설함
- **삼전법륜(三轉法輪):** 삼(셋 삼). 유식(唯識) — 공(空)을 넘어 삼성(三性)의 진실을 설함. 가장 완전한 가르침

---

## 용어 설명

- **유식(唯識)** — 유(오직 유)·식(알 식). 오직 마음의 식(識)만이 있음. 외부 세계는 마음이 만들어낸 것
- **아뢰야식(阿賴耶識)** — 아(저 아)·뢰(의지할 뢰)·야(집 야)·식(알 식). 모든 경험과 업(業)의 씨앗을 저장하는 근본 식(識). 창고(藏)와 같은 마음
- **삼성(三性)** — 삼(셋 삼)·성(성품 성). 변계소집성(遍計所執性 — 두루 변·헤아릴 계·바 소·잡을 집·성품 성)·의타기성(依他起性 — 의지할 의·다를 타·일어날 기·성품 성)·원성실성(圓成實性 — 둥글 원·이룰 성·참 실·성품 성)
- **삼무자성(三無自性)** — 삼(셋 삼)·무(없을 무)·자(스스로 자)·성(성품 성). 세 가지 측면에서 자성(自性)이 없음. 공(空) 사상의 유식적 해석

---

## 핵심 개념 해설

- 제1품 서품(序品) — 서(차례 서)·품. 경전의 서문(序文)과 설법 배경
- 제2품 승의제상품(勝義諦相品) — 승(이길 승)·의(뜻 의)·제(진리 제)·상(모양 상)·품. 승의(勝義 — 뛰어난 진리)의 모습 — 언어와 개념을 초월한 진실
- 제3품 심의식상품(心意識相品) — 심(마음 심)·의(뜻 의)·식(알 식)·상(모양 상)·품. 마음(心)·뜻(意)·식(識)의 모습 — 아뢰야식(阿賴耶識) 설명
- 제4품 일체법상품(一切法相品) — 일(하나 일)·체(몸 체)·법(법 법)·상(모양 상)·품. 삼성(三性) — 변계소집성(遍計所執性)·의타기성(依他起性)·원성실성(圓成實性)
- 제5품 무자성상품(無自性相品) — 무(없을 무)·자(스스로 자)·성(성품 성)·상(모양 상)·품. 삼무자성(三無自性) — 모든 법에는 고정된 자성(自性)이 없다
- 제6품 분별유가품(分別瑜伽品) — 분(나눌 분)·별(다를 별)·유(맬 유)·가(더할 가)·품. 유가행(瑜伽行 — 맬 유·더할 가·행할 행) 수행 — 지(止)와 관(觀)
- 제7품 지바라밀다품(地波羅蜜多品) — 지(땅 지)·바(물결 파)·라(그물 라)·밀(빽빽할 밀)·다(많을 다)·품. 십지(十地)와 바라밀(波羅蜜 — 완성)의 수행
- 제8품 여래성소작사품(如來成所作事品) — 여(같을 여)·래(올 래)·성(이룰 성)·소(바 소)·작(지을 작)·사(일 사)·품. 여래(如來)의 법신(法身)·보신(報身)·화신(化身) 삼신(三身)

---

## 경전의 의의

해심밀경은 유식학파(唯識學派)의 근본 경전으로, 아뢰야식(阿賴耶識)과 삼성(三性)·삼무자성(三無自性)을 통해 "모든 것은 마음이 만든다"는 유식(唯識) 사상을 체계화합니다. 삼전법륜(三轉法輪)을 통해 부처님의 가르침을 아함·반야·유식으로 구분하는 불교 교판(敎判)의 근거가 됩니다.
