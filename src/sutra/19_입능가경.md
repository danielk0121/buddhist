# 입능가경 (入楞伽經)

입(들 입)·능(능할 능)·가(더할 가)·경(경전 경)

> 입능가경(入楞伽經)
> 북위(北魏) 보리류지(菩提流支 — 보살 보·끌 제·흐를 류·가를 지) 한역 (513년) | 현대 우리말 풀어쓰기

---

## 경전 소개

- **원어:** 산스크리트어 원전 『랑카바타라 수트라(Laṅkāvatāra Sūtra)』를 한문으로 번역. 유명 한역본으로 구나발타라(求那跋陀羅) 역 『능가아발다라보경(楞伽阿跋多羅寶經)』 4권과 실차난타(實叉難陀) 역 『대승입능가경(大乘入楞伽經)』 7권이 있음
- **구성:** 10권 18품(品 — 품 품)
- **핵심 사상:** 여래장(如來藏)과 아뢰야식(阿賴耶識)의 관계. 오법(五法 — 다섯 오·법 법)·삼자성(三自性 — 셋 삼·스스로 자·성품 성)·팔식(八識 — 여덟 팔·알 식)·이무아(二無我 — 둘 이·없을 무·나 아). 언어와 개념을 초월한 내증(內證 — 안 내·증거 증)의 세계
- **특징:** 선종(禪宗 — 선 선·종파 종)의 초조(初祖 — 처음 초·할아버지 조) 달마대사(達磨大師 — 통할 달·갈 마·클 대·스승 사)가 혜가(慧可 — 지혜 혜·옳을 가)에게 전한 경전. 유식(唯識)과 여래장 사상을 통합

---

## 원문 (한문)

```
一切法無我 無生無自性
如幻如夢境 此是諸佛說
(입능가경 게송 — 자내증의 핵심 가르침)
```

## 18품 전체 목록

- 제1품 나바나왕권청품(羅婆那王勸請品) — 나(그물 나)·바(할멈 파)·나(어찌 나)·왕(임금 왕). 능가산(楞伽山)의 왕 라바나(羅婆那)가 부처님께 설법을 청함
- 제2품 집일체불법품(集一切佛法品) — 집(모을 집)·일(하나 일)·체(몸 체)·불(부처 불)·법(법 법)·품. 모든 불법(佛法)의 핵심
- 제3품 무상품(無常品) — 무(없을 무)·상(항상할 상)·품. 무상(無常)의 의미
- 제4품 현증품(現證品) — 현(나타날 현)·증(증거 증)·품. 내증(內證)의 경지
- 제5품 여래상주품(如來常住品) — 여(같을 여)·래(올 래)·상(항상할 상)·주(머물 주)·품. 여래는 항상 머문다
- 제6품 찰나품(刹那品) — 찰(절 찰)·나(어찌 나)·품. 찰나(刹那 — 매우 짧은 시간)와 무상(無常)
- 제7품 변화품(變化品) — 변(변할 변)·화(될 화)·품. 변화와 환상
- 제8품 일천제품(一闡提品) — 일(하나 일)·천(열 천)·제(끌 제)·품. 불성(佛性)이 없다고 여겨지는 일천제
- 제9품 여래장품(如來藏品) — 여(같을 여)·래(올 래)·장(감출 장)·품. 여래장(如來藏)의 의미와 아뢰야식(阿賴耶識)의 관계
- 제10품 현식품(現識品) — 현(나타날 현)·식(알 식)·품. 현상을 드러내는 식(識)
- 제11품 오법품(五法品) — 오(다섯 오)·법(법 법)·품. 오법(五法) — 명(名)·상(相)·분별(分別)·정지(正智)·여여(如如)
- 제12품 성문품(聲聞品) — 성(소리 성)·문(들을 문)·품. 성문(聲聞 — 소리를 듣는 자)의 수행
- 제13품 찰토장엄품(刹土莊嚴品) — 찰(절 찰)·토(흙 토)·장(꾸밀 장)·엄(엄숙할 엄)·품. 불국토(佛國土)의 장엄
- 제14품 불성품(佛性品) — 불(부처 불)·성(성품 성)·품. 불성과 여래장
- 제15품 수행품(修行品) — 수(닦을 수)·행(행할 행)·품. 수행의 방법과 단계
- 제16품 게송품(偈頌品) — 게(게 게)·송(읊을 송)·품. 게송(偈頌)으로 핵심을 요약
- 제17품 총품(總品) — 총(모을 총)·품. 전체 내용 요약
- 제18품 다라니품(陀羅尼品) — 다(많을 다)·라(그물 라)·니(진흙 니)·품. 다라니(陀羅尼 — 총지 總持)

## 핵심 사상

### 오법 (五法)

오(다섯 오)·법(법 법)

- **명(名):** 이름 명 — 언어와 개념으로 붙인 이름
- **상(相):** 모양 상 — 이름에 대응하는 모양과 현상
- **분별(分別):** 나눌 분·다를 별 — 이름과 모양을 나누어 구별하는 작용
- **정지(正智):** 바를 정·알 지 — 올바른 지혜로 분별을 초월함
- **여여(如如):** 같을 여·같을 여 — 있는 그대로의 진실. 언어와 분별을 넘어선 경지

### 팔식 (八識)

팔(여덟 팔)·식(알 식)

- 안(眼 — 눈 안)·이(耳 — 귀 이)·비(鼻 — 코 비)·설(舌 — 혀 설)·신(身 — 몸 신)·의(意 — 뜻 의) 육식(六識)
- 말나식(末那識 — 끝 말·어찌 나·알 식) — 제7식(識). 자아(自我)에 집착하는 식
- 아뢰야식(阿賴耶識) — 제8식(識). 모든 종자(種子 — 씨앗)를 저장하는 근본 식

### 자내증 (自內證)

자(스스로 자)·내(안 내)·증(증거 증)

- 언어(言語)와 문자(文字)·개념(槪念)을 초월한 스스로의 내면 증득(證得 — 증거 증·얻을 득)
- 선종(禪宗)의 불립문자(不立文字 — 아닐 불·설 립·글 문·글자 자)와 이심전심(以心傳心 — 써 이·마음 심·전할 전·마음 심)과 맥락을 같이 함

## 핵심 개념 해설

- **능가(楞伽)** — 능(능할 능)·가(절 가). 산스크리트어 Laṅkā의 음역. 지금의 스리랑카(Sri Lanka). 경전의 설법지(說法地)
- **여래장(如來藏)** — 여(같을 여)·래(올 래)·장(감출 장). 번뇌에 가려진 본래의 청정한 불성(佛性). 아뢰야식(阿賴耶識)과의 관계를 설명
- **아뢰야식(阿賴耶識)** — 아(저 아)·뢰(의지할 뢰)·야(집 야)·식(알 식). 모든 경험의 씨앗을 저장하는 근본 식(識). 여래장과 깊이 연관됨
- **자내증(自內證)** — 자(스스로 자)·내(안 내)·증(증거 증). 스스로 내면에서 증득(證得)하는 것. 언어와 개념을 넘어선 직접적인 깨달음. 선종(禪宗)의 불립문자(不立文字)와 통함
- **불립문자(不立文字)** — 불(아닐 불)·립(설 립)·문(글 문)·자(글자 자). 문자(文字)에 의존하지 않음. 직접적인 깨달음을 강조하는 선종의 태도

---

## 용어 설명

- **능가(楞伽)** — 능(능할 능)·가(절 가). 산스크리트어 Laṅkā의 음역. 지금의 스리랑카. 경전의 설법지
- **여래장(如來藏)** — 여(같을 여)·래(올 래)·장(감출 장). 번뇌에 가려진 본래의 청정한 불성
- **아뢰야식(阿賴耶識)** — 아(저 아)·뢰(의지할 뢰)·야(집 야)·식(알 식). 모든 경험의 씨앗을 저장하는 근본 식
- **자내증(自內證)** — 자(스스로 자)·내(안 내)·증(증거 증). 언어와 개념을 넘어 스스로 내면에서 증득하는 것
- **불립문자(不立文字)** — 불(아닐 불)·립(설 립)·문(글 문)·자(글자 자). 문자에 의존하지 않음. 선종의 직접 체험 강조
- **팔식(八識)** — 팔(여덟 팔)·식(알 식). 안·이·비·설·신·의 육식에 말나식·아뢰야식을 더한 여덟 가지 의식

---

## 경전의 의의

입능가경은 여래장(如來藏) 사상과 유식(唯識) 사상을 결합하며 선종(禪宗)에 깊은 영향을 미친 경전입니다. 달마 대사가 중국에서 선종을 전할 때 이 경전을 근거로 삼았으며, "자내증(自內證)"의 강조는 언어와 개념을 넘어선 직접 체험을 중시하는 선 수행 전통의 근거가 됩니다.
